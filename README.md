# 🚀 DockDeck

Windows向けのDock風ランチャーアプリです。

Electronを使用して開発し、URL起動・ボタン追加・透明度変更などの機能を実装しました。

---

## ✨ 主な機能

- URLランチャー
- ボタン追加
- ボタン削除
- 設定保存
- 透明度変更
- 横表示 / 縦表示切替
- 常に最前面表示
- シンプルなDock風UI

---

## 🛠 使用技術

| 技術 | 内容 |
|--------|--------|
| Electron | デスクトップアプリ開発 |
| Node.js | 実行環境 |
| JavaScript | ロジック実装 |
| HTML | UI構築 |
| CSS | デザイン |

